# Adoption Guides

Adoption guides are non-normative implementation aids.

They explain how an institution might apply the Governance Constitution and relevant domain profiles in practice.

They do not amend the Constitution, certify compliance, or replace domain law.

## Available guides

- `court_or_tribunal_adoption.md` — early draft guide for courts, tribunals, and court-adjacent institutions.

## Planned guides

- public sector adoption;
- hospital adoption;
- university adoption;
- financial institution adoption.
