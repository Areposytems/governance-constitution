# Academic Domain Profile

Status: Placeholder draft v0.1  
Compatible with: Governance Constitution v1.0  
Type: Non-normative domain translation profile

## Purpose

This profile will translate the Governance Constitution into academic, educational, research, and scholarly settings.

It should address:

- research integrity;
- citation integrity;
- student AI use;
- assessment;
- authorship;
- peer review;
- data provenance;
- institutional knowledge systems;
- academic misconduct processes;
- transparent and contestable AI-assisted evaluation.

## Boundary

This profile does not replace university policy, research ethics approval, disciplinary process, or academic judgment.
