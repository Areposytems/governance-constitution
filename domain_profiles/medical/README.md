# Medical Domain Profile

Status: Placeholder draft v0.1  
Compatible with: Governance Constitution v1.0  
Type: Non-normative domain translation profile

## Purpose

This profile will translate the Governance Constitution into medical and health-domain language.

It should address:

- patient safety;
- clinician responsibility;
- informed consent;
- diagnostic support;
- treatment support;
- health data privacy;
- triage;
- adverse-event reporting;
- review and contestability;
- clinical governance.

## Boundary

This profile does not create medical compliance and does not replace clinical judgment, professional duties, health law, hospital governance, or therapeutic responsibility.
