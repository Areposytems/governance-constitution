# Jurisdictional Adaptation Note v0.1

## Purpose

The Legal Domain Profile is jurisdiction-neutral. This note helps adopting institutions translate the profile into local legal terminology without changing the constitutional core.

## Core rule

Adopt the function, not merely the wording.

If a term in the profile does not match local law, the adopting institution should identify the equivalent local concept, office, duty, process, or safeguard.

## Common translation pairs

|Jurisdiction-neutral term|Possible local equivalents|
|-|-|
|Procedural fairness|Due process, natural justice, fair hearing, administrative fairness|
|Legal practitioner|Lawyer, attorney, solicitor, barrister, advocate, counsel, legal representative|
|Protected confidentiality|Legal professional privilege, attorney-client privilege, litigation privilege, professional secrecy, confidential legal communication|
|Rights-affecting decision|Decision affecting rights, duties, entitlements, benefits, liberty, legal status, sanctions, obligations, access to services, or procedural treatment|
|Adjudicative body|Court, tribunal, commission, panel, board, authority, ombuds institution, administrative decision-maker|
|Sworn or affirmed statement|Affidavit, declaration, witness statement, deposition, certified statement, formally submitted evidence|
|Human reconsideration|Review, appeal, objection, correction process, complaint pathway, redetermination, reconsideration by an authorised person|
|Authorisation record|Approval record, governance artifact, delegated authority record, decision log, control record, legitimacy binding artifact|

## Local adoption questions

Before adoption, the institution should answer:

1. Which local legal duties does this profile support?
2. Which local roles remain accountable for AI-assisted work?
3. Which AI uses require disclosure?
4. Which AI uses require formal authorisation before reliance?
5. Which AI uses are prohibited unless expressly authorised by law or institutional rule?
6. Which affected persons must be able to challenge, correct, review, or seek human reconsideration?
7. How long does an authorisation remain valid before review or expiry?
8. Which records must be retained, and which should not be retained for confidentiality, privilege, privacy, or security reasons?

## Boundary

This note does not create legal compliance. It supports translation between the Governance Constitution and local legal systems.
